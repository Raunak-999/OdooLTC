#### **8. Upvote Arrow Buttons (MISSING - Critical for Engagement)**
**Location**: Left side of each question card, above the vote count
**Required Functionality**:
- **Visual Design**: ChevronUpIcon from Heroicons, green when active
- **Click Action**: Increase question vote count by 1
- **Authentication Check**: Must be logged in to vote
- **Vote Validation**: Cannot vote on own questions
- **State Management**:

  - Green when user has upvoted
  - Disabled state if user already downvoted (must remove downvote first)
- **Database Update**: Update vote count in Firestore immediately
- **Real-time Updates**: All users see vote count change instantly
- **Error Handling**: Show error if vote fails
- **Loading State**: Brief spinner during vote processing

#### **9. Downvote Arrow Buttons (MISSING - Critical for Engagement)**

- **Authentication Check**: Must be logged in to vote
- **Vote Validation**: Cannot vote on own questions
- **State Management**:
  - Gray when not voted
  - Red when user has downvoted
  - Disabled state if user already upvoted (must remove upvote first)
- **Database Update**: Update vote count in Firestore immediately
- **Real-time Updates**: All users see vote count change instantly

- **Error Handling**: Show error message if vote fails
